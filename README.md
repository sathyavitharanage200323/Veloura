# ✨ Veloura

**Luxury Event Planning & Vendor Management Platform**

Veloura is a modern web-based platform that simplifies planning and managing luxury events. Whether it's a wedding, birthday, corporate event, anniversary, or private celebration, Veloura connects customers with trusted vendors and provides all the tools needed to create unforgettable experiences.

## 🌟 Features

- 💍 Wedding Planning
- 🎂 Birthday Event Management
- 🏢 Corporate Event Planning
- 🎉 Anniversary Celebrations
- 👶 Baby Shower Planning
- 📍 Venue Booking
- 🍽️ Catering Services
- 📸 Photographer & Videographer Booking
- 🌸 Decoration Services
- 🎵 Entertainment & DJ Booking
- 💳 Secure Online Payments
- 📅 Event Scheduling
- 👥 Guest Management
- ⭐ Vendor Reviews & Ratings

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS

### Backend
- Laravel
- REST API

### Database
- MySQL
